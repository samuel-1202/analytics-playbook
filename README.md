# Analytics Playbook

개인 학습 및 실무 경험을 바탕으로 정리한 Data Analytics / Experimentation / Machine Learning 지식 저장소입니다.

단순 개념 정리를 넘어, 실제 실험 설계·분석 코드·노트북 예제를 함께 관리하며 지속적으로 업데이트하고 있습니다.

# Goal

이 저장소의 목표는 단순한 지식 정리가 아니라,

- 실험 설계 역량 향상

- 머신러닝 이해도 향상
- 추천 시스템 이해도 향상
- Data Scientist 역량 확보

를 위한 개인 플레이북 구축입니다.
